<h1 align="center" id="title">todo app</h1>

<p id="description">A basic todo app built using flask and mongodb.</p>

<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repo</p>

```
git clone https://github.com/Sundaram-2001/flask-todo.git
```

<p>2. Install Flask</p>

```
pip install flask
```

<p>3. Install pymongo</p>

```
pip install pymongo
```

<p>4. To run the application</p>

```
python3 app.py
```

Technologies used in the project:
